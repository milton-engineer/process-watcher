# Process Watcher

Linux-based process monitoring and inspection utility.

## Overview

Process Watcher is a lightweight command-line tool designed to inspect running Linux processes and assist in troubleshooting system resource usage.

The project simulates operational workflows commonly used in infrastructure monitoring and backend support environments.

## Features

- Running process inspection
- CPU usage monitoring
- Memory consumption analysis
- Lightweight terminal interface
- Linux troubleshooting support

## Technologies

- Python
- Linux

## Goals

- Practice Linux process monitoring
- Simulate infrastructure diagnostics
- Build lightweight observability tools
- Improve operational troubleshooting workflows
